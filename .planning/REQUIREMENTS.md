# Requirements: v1.11 Reader Experience & Creator Discovery

Defined: 2026-04-09
Core Value: Connect readers to creators while delivering a focused chapter reading experience with reliable progression sync.

## v1.11 Requirements

### Author Profiles

- [ ] AUTHOR-01: Reader can view a public author profile page with core creator information.
- [ ] AUTHOR-02: Reader can view an author's published novel catalog from the profile page.
- [ ] AUTHOR-03: Reader can view aggregate creator statistics (for example total views across published novels).

### Reader Interface

- [ ] READER-01: Reader can open a distraction-free chapter reading UI optimized for long-form text.
- [ ] READER-02: Reader can navigate chapters with next/previous controls and table-of-contents jumps.

### Reader Customization

- [ ] READER-03: Reader can adjust font size in-reader and preference applies immediately.
- [ ] READER-04: Reader can toggle light/dark reading mode in-reader and preference applies immediately.

### Progression Sync

- [ ] SYNC-01: Reader chapter opens safely trigger backend view count updates.
- [ ] SYNC-02: Reader position and history update so authenticated users can resume without losing place.

## Future Requirements (Deferred)

- CREATOR-01: Author follower counts and social graph insights on profile page.
- READER-05: Advanced typography controls (line height, font family, content width).
- SYNC-03: Cross-device real-time resume synchronization.

## Out of Scope (v1.11)

| Feature | Reason |
|---------|--------|
| Native mobile app reader UI | Web milestone focus |
| Embedded social/reaction overlays in reader | Keep chapter reading distraction-free for this milestone |
| ML-based recommendation ranking within reader | Defer until core reader journey is stable |
