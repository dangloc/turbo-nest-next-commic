# Roadmap: v1.16 – The Revenue Pipeline – SePay Integration & User Top-Ups

## Overview
Build a complete end‑to‑end top‑up system, migrating legacy PHP SePay webhook logic to NestJS/Prisma, exposing a reader‑facing top‑up UI, and wiring the admin wallet dashboard to live data.
