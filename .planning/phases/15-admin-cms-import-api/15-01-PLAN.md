  <action>
Create a `CmsModule` with controller + service and wire it into `AppModule`.

Controller requirements:
- Route: `POST /admin/import`.
- Guards: `@UseGuards(RolesGuard)` and `@Roles('ADMIN')`.
- Upload: `FileInterceptor('file')` with multer filter allowing only `.txt` and `.docx`.

Service requirements:
- Call parser service and validate output.
- Persist novel + chapters in a single Prisma transaction.
- Novel: compute next integer id, set `uploaderId` from authenticated admin, set `title` and raw/full `postContent`.
- Chapters: compute sequential ids, map parser output to chapter records, `createMany` against chapter table.
- Return `{ novelId, chaptersCreated, errors, warnings }` where `chaptersCreated` includes chapterNumber/title/id.

Use defensive error handling for empty files, parser failures, and DB write failures.
