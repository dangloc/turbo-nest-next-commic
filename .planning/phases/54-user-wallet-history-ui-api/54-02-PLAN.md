<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser -> API | User session and pagination query are untrusted inputs. |
| API response -> UI | Remote financial data is rendered into user-visible table/list cells. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-54-05 | Information Disclosure | `/profile/wallet` UI | mitigate | Client only calls self-scoped endpoint; unauthenticated users see login CTA. |
| T-54-06 | Tampering | page/pageSize query construction | mitigate | Clamp numeric values in API wrapper and backend. |
| T-54-07 | XSS | transaction description/content rendering | mitigate | Render as React text only; never use `dangerouslySetInnerHTML`. |
| T-54-08 | Denial of Service | repeated refresh/pagination | mitigate | Use bounded pageSize, abort controllers, and no polling loop. |
</threat_model>
