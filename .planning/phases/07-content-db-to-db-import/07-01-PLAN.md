# Migration Function Pattern (from v1.0)

All migrate-* functions follow this structure:
```typescript
export async function migrateX(
  rows: SourceXRow[],
  deps: { repo: XRepo; ... }
): Promise<{ xUpserted: number; ... }> {
  let xUpserted = 0;
  for (const row of rows) {
    await deps.repo.upsert(row);
    xUpserted++;
  }
  return { xUpserted };
}
```

Content migration should follow the same pattern—no quarantine needed (content errors don't affect users).
